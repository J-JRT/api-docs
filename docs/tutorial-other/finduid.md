# Find uid Facebook

```jsx title="API Endpoint:"
    http://mzkapi.me/finduid?url=<url profile, story, post, video,...>
```
## Return data
```jsx title="http://mzkapi.me/finduid?url=http://www.facebook.com/duonggg216"
{
    "success": 200,
    "id": 100049354060779,
    "author": "D-Jukie"
}
```
