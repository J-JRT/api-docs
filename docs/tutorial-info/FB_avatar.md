# Get avatar Facebook

```jsx title="API Endpoint:"
    http://mzkapi.me/facebook/avatar?uid=`UID`&apikey=`API_KEY`
```
## Return data

{
    "author": "MạnhG",
    "data": {
        "name": "Dương Dương",
        "id": "100049354060779",

}
